# Attendance App (PyQt6)

## Description

This is a simple attendance tracking app I made using Python and PyQt6.
You can enter a student’s name and mark them as present or absent.
The data gets saved to a CSV file so it stays even after you close the app.

## Features

* Enter a student name
* Mark present or absent
* Saves data to a file
* Simple GUI

## How to Run

1. Install PyQt6
   pip install PyQt6

2. Run the program
   python main.py
